## project에 포함시킬 기능
- **일정에 대한 CRUD 기능** : 각 날짜별로 일정을 추가, 읽기, 수정, 삭제를 할 수 있다.
- **일정 분류** : 각 일정에 태그를 추가하여 원하는 태그의 일정을 확인할 수 있다.
- **일정 알림** : 프로그램 실행시 금일에 해당하는 일정을 알려준다.
- **일정 검색** : 일정에 대한 키워드를 입력하여 원하는 내용의 일정을 검색할 수 있다.
- **일정 관리** : 완료 여부 체크, 중요한 일정 등을 표시할 수 있다.
